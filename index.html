Copilot Agent mode

  • Task-focused assistant inside your editor
  • Plans steps, edits multiple files, runs tasks/tests
  • Grounded in your workspace context
  • You stay in control: review, approve, iterate

Agent mode: example


Goal: Add a /health endpoint that returns build info and status

Better agent prompts

  • Goal + constraints + acceptance criteria
  • Point to files/folders when possible
  • Prefer small scopes; iterate quickly
  • Ask the agent to show diffs and test results
